# OCP_ALADIN

## Introduction

`OCP_ALADIN` is an interface for solving optimal control problems by
"Augmented Lagrangian based Alternating Direction Inexact Newton" method
([ALADIN](http://sist.shanghaitech.edu.cn/faculty/boris/ALADIN.html)).

## Dependencies

* CasADi
* MATLAB
